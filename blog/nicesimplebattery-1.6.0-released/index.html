Nice Simple Battery 1.6.0 Released

Nice Simple Battery has been updated to 1.6.0. Now you can draw the small battery cap on top of your battery icon. It looks like this:

Additionally, there is a new option that allows you to use a different color to draw your battery icon when the battery power level is below a specified level. This should be useful to remind yourself to charge your battery when the battery is running low.

Hope you would like these new feature in this update. Here is the changelog:

Version 1.6.0:

- New feature: Option to draw the battery cap (under the "Battery Icon" section)
- New feature: Options for using a different color when battery is low (under the "Advanced Settings" section)
- Now clicking on the preview can change the power level displayed in the preview
- Applied a number of bug fixes and minor changes
