Nice Simple Clock 1.8.0 Released

This week, I have updated the Nice Simple Clock. The new "year offset" feature is for special calendar systems that are being used in certain places around the world. Also, if you would like to see 13 instead 2013 for the year display, you can also subtract 2000 from the year using this new option.

Additionally, I have added a few new time formats and new date separators in this update.

Here is the changelog:

Version 1.8.0:

- New feature: Custom year offset option for special calendar systems (under "Other Settings")
- Added 3 new time formats and 3 new date separators
- Applied a number of bug fixes and minor changes
