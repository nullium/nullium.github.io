StyleNote《2.0 版》

StyleNote《2.0 版》發佈了！

在這個重大更新中，加入了很多新功能及改進。主介面也重新設計過了。



主要的新功能包括：強化了的備份功能、記事搜尋功能、列點清單的支援、Style 樣板及 Dropbox 雲端儲存的支援。我現在太累了，所以不會為它們逐一詳細介紹。但你可在本篇網誌的結尾找到完整的改版資訊。

舊用家可到設定頁查看最新的使用說明，特別是列點清單的功能介紹。設定頁可在主選單中找到，而主選單則可透過程式右上方的「三小點」圖示打開。其餘大部份的新功能應該不難使用吧。(希望是這樣吧 :))

如果你有遇到甚麼問題，請告訴我知道。你也可以到新設的用戶論壇查看常見問題、發問及討論。

希望你會喜歡新一代的 StyleNote。 :)

《2.0.0 版》

- 新功能：可自定義範圍的記事搜尋功能
- 新功能：支援在用家的 Dropbox (免費雲端網絡儲存)上下載記事及設定以進行雲端備份或不同裝置間的資料同步
- 新功能：可滙出記事及設定到 HTML 檔，可在任何電腦上直接開啟閱讀
- 新功能：可從滙出的 HTML 檔滙入記事及設定
- 新功能：支援有加密的滙出及滙入 (SNB2 檔)
- 新功能：定期自動備份功能 (Pro 進階版用家可用 Dropbox 作定期備份及自選備份頻率)
- 新功能：自動插入清單列點功能
- 新功能：一鍵就可在列點上劃上刪除線的功能
- 新功能：可把記事的自訂顏色及大小儲存到 Style 樣版供日後讀取套用
- 新功能：現在可把便條小工具的文字置中
- 新功能：現在在萬一遺失密碼的時候也可透過回答自設的私密問題重設密碼
- 新功能：現在按記事列表上的網址超連結可直接打開瀏覽器
- 新功能：現在可以在其他程式傳送或分享文字到 StyleNote
- 重新設計了主介面
- 還有很多其他的改進和小改動
- StyleNote 現在需要 Android 版本 2.1 或以上才可運行 (Android 1.5 和 1.6 仍可使用 StyleNote 1.6.3 版，可到 nullium.com/hk 下載)
